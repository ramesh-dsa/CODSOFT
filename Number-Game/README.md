# 🎯 Number Guessing Game using Java

A simple and interactive console-based Number Guessing Game developed using Java.  
The program generates a random number, and the user must guess the correct number with the help of hints like "Too High" or "Too Low".

---

## ✨ Features

✅ Random Number Generation  
✅ User Guess Input  
✅ Hint System (Too High / Too Low)  
✅ Attempt Counter  
✅ Interactive Console Output  
✅ Loop-Based Gameplay  

---

## 🛠️ Technologies Used

- ☕ Java
- 🎲 Random Class
- 📥 Scanner Class
- 🔁 Loops
- 🔀 Conditional Statements

---

## 📌 Project Description

This project was developed as part of a Java Programming Internship task.  
The game randomly generates a number between 1 and 100, and the player continues guessing until the correct number is found.

The program also tracks the number of attempts taken by the user to guess the correct number.
