# 🏧 ATM Interface using Java

A simple and interactive **console-based ATM Interface** developed using **Java**.  
This project simulates basic ATM operations such as depositing money, withdrawing money, and checking account balance through a menu-driven system.

---

## ✨ Features

✅ Deposit Money  
✅ Withdraw Money  
✅ Check Account Balance  
✅ Input Validation  
✅ Insufficient Balance Handling  
✅ Menu-Driven Interface  
✅ Continuous User Interaction using Loops  

---

## 🛠️ Technologies Used

- ☕ Java
- 📥 Scanner Class
- 🔁 Loops
- 🔀 Conditional Statements
- 🧠 Programming Logic

---

## 📌 Project Description

This project was developed as part of a Java Programming Internship task.  
The ATM system allows users to perform basic banking operations in a simple and user-friendly way using terminal/console interaction.

The program also handles invalid inputs and prevents users from withdrawing money beyond the available balance.
