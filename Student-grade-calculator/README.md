# 📘 Student Grade Calculator using Java

A simple and efficient console-based Student Grade Calculator developed using Java.  
This project calculates the total marks, average percentage, and grade of a student based on the marks entered by the user.

---

## ✨ Features

✅ Dynamic Subject Input  
✅ Total Marks Calculation  
✅ Average Percentage Calculation  
✅ Grade Evaluation System  
✅ User-Friendly Console Output  
✅ Loop-Based Input Handling  

---

## 🛠️ Technologies Used

- ☕ Java
- 📥 Scanner Class
- 🔁 Loops
- 🔀 Conditional Statements
- 📊 Arrays

---

## 📌 Project Description

This project was developed as part of a Java Programming Internship task.  
The program allows the user to enter marks for multiple subjects dynamically and automatically calculates:

- Total Marks
- Average Percentage
- Final Grade

The grading system is implemented using conditional statements based on the calculated average percentage.
